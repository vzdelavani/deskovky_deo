Výběr deskových her pro knihovny (testovací verze)
Jednoduchý konzolový nástroj pro knihovníky, který pomáhá doporučit deskové hry podle:

věkové skupiny
délky hraní
stylu hry
tématu
Jak spustit
Je potřeba mít nainstalovanou Javu (verze 17+).

Stáhni soubor GameSelectorConsole.java
Otevři terminál ve složce se souborem
Spusť:
javac GameSelectorConsole.java
java GameSelectorConsole
